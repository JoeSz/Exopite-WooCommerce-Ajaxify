DESCRIPTION
-----------

WordPress Plugin to add amount for single product on shop and ajaxify add to cart button on product pages.

REQUIREMENTS
------------

Server

* WordPress 4.7+ (May work with earlier versions too)
* PHP 5.3+ (Required)
* jQuery 1.9.1+

Browsers

* Modern Browsers
* Firefox, Chrome, Safari, Opera, IE 10+
* Tested on Firefox, Chrome, Edge, IE 11

INSTALLATION
------------

1. [x] Upload `exopite-lightbox35` to the `/wp-content/plugins/exopite-lightbox35/` directory
